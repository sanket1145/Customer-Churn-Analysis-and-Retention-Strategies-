# Customer Churn Analysis & Retention Strategy Dashboard

## Business Problem
Telecom companies face revenue loss due to customer churn. This project analyzes customer behavior to identify churn drivers and propose data-driven retention strategies.

## Role of Data Analyst
- Understand churn problem
- Analyze customer data
- Identify churn patterns
- Build dashboards
- Recommend retention strategies

## Dataset
- Source: Kaggle (Telco Customer Churn)
- Records: 7,000+ customers

## Tools Used
Python, SQL (MySQL), Power BI, Excel

## Workflow
1. Data cleaning and EDA in Python
2. SQL analysis for churn insights
3. Power BI dashboard for storytelling

## Key Insights
- Month-to-month contracts have highest churn
- New customers churn more
- Fiber optic users show higher churn
- High monthly charges increase churn risk

## Retention Strategies
- Long-term contract discounts
- Loyalty benefits after 12 months
- Auto-payment incentives
- Improve fiber service reliability
- Personalized pricing plans
